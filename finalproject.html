Final Project - Raining Rocks

This is my game that I programmed using the code language javascript. I have used all of the different things that I have learned from this class and more, to be able to accomplish this project. This game consist starts off with the title screen instructing you how to play. This game you are a cute character and you must use the LEFT or RIGHT arrow keys to avoid the falling rocks. If you are hit with the rocks you are deducted 100 points. Every second you earn 100 points, reach 1000 points to win. You are timed, so try to beat you best record.
